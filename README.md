# sample-ws
# Overview 

This is an authentication web project, using the Go language with the JWT authentication method. 
This is part of my journey of learning back-end development.

# Features 

1. /auth authenticates the user login and returns a token
2. Uses the retrieved token and generates a simple SQL query.
